
W3m usage:

1. 使用Cookie在W3m中:
  w3m -cookie

2. 使用标签页:
   T		开新的空白标签页.
   C-t		在新标签页里打开链接.
   C-q		关闭标签页.
   ESC-t	显示所有的标签页.
   }	 	跳到下一个标签页.
   {	 	跳到前一个标签页.

3. 浏览网页:
   U		输入URL, 并且打开URL.
   B		返回上一个页面.
   ESC-m	列出页面中的所有超链接.
   ESC-g	输入行号并且跳到该行的头部.
   [		跳到页面的第一个超链接.
   ]		跳到页面的最后一个超链接.
   L		显示所有的超链接和它们的显示文字.
   I		显示超链接指向的图片.
   v		显示/关闭显示页面源代码.
   R		刷新页面.
